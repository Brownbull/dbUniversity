Name    Path         Http Verb  Purpose
======================================
Index   /dogs              GET     List all dogs
New     /dogs/new          GET     Show new dog form
Create  /dogs     ->Rdrct  POST    Create a new dog, then REDIRECT somewhere
Show    /dogs/:id          GET     Show info about one specific dog
Edit    /dogs/:id/edit     GET     Show edit form for one dog
Update  /dogs/:id ->Rdrct  PUT     Update a particular dog, then REDIRECT somewhere
Destroy /dogs/:id ->Rdrct  DELETE  Delete a particular dog, then REDIRECT somewhere
